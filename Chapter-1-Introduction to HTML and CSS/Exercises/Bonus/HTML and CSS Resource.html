Basic HTML Tags

<h1> to <h6> - Headings

From largest (<h1>) to smallest (<h6>), this is used to identify headers.

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p> - Paragraph

Specifies a text block.

<p>Paragraph</p>

<figure> and <figcaption> - Multimedia Content

Used for information that stands alone, such as pictures with descriptions.

<figure>
  <img src="image.jpg" alt="Description" />
  <figcaption>Caption</figcaption>
</figure>
